# Storage Manager System
Aplicação web para gestão de estoque e vendas voltada a pequenos e médios
empreendedores do setor alimentício. Permite controle de insumos, saídas,
vendas e funcionários com sistema de permissões por perfil de usuário.

## Tecnologias
| Tecnologia | Uso |
|---|---|
| React 18 | Construção da interface |
| Redux + Redux Saga | Gerenciamento de estado e side effects assíncronos |
| Redux Persist | Persistência de sessão no localStorage |
| Styled Components | Estilização com CSS-in-JS |
| Axios | Requisições HTTP à API REST |
| Chart.js + react-chartjs-2 | Visualização de dados no dashboard |
| React Router DOM | Navegação e rotas protegidas |
| React Toastify | Feedback visual de ações |

# Storage Manager System
Web application for inventory and sales management aimed at small and medium-sized entrepreneurs in the food sector. Allows control of supplies, outputs, sales and employees with a permissions system by user profile.

| Technology | Usage |
|---|---|
| React 18 | Interface Construction |
| Redux + Redux Saga | Asynchronous State Management and Side Effects |
| Redux Persist | Session Persistence in LocalStorage |
| Styled Components | Styling with CSS-in-JS |
| Axios | HTTP Requests to the REST API |
| Chart.js + react-chartjs-2 | Data Visualization on the Dashboard |
| React Router DOM | Navigation and Protected Routes |
| React Toastify | Visual Feedback of Actions |

## Features
- Authentication with JWT token and session persistence
- Dashboard with sales, input, and output graphs by period
- Complete CRUD for inputs, outputs, sales, and employees
- Access control by profile (administrator and multiple employee levels)
- Search and filter records by different parameters
- Real-time inventory monitoring with automatic updates
